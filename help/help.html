Справочная система "Угольная Шахта"

1. Назначение системы

Система предназначена для автоматизации учета и управления на угольной шахте.

Основные функции:

  • Управление справочной информацией
  • Учет добычи угля и вывоза породы
  • Контроль затрат ресурсов
  • Планирование и анализ выполнения планов
  • Формирование отчетности

2. Структура меню

Файл

  • Экспорт данных - сохранение данных в формате Excel
  • Выход - закрытие приложения

Справочники

  • Уголь - характеристики марок угля
  • Участки - информация о производственных участках
  • Должности - список должностей работников
  • Работники - учет сотрудников шахты

Операции

  • Добыча - ежедневный учет добычи угля
  • Затраты - учет расходов электроэнергии и топлива
  • Учет времени - табель рабочего времени

Планирование

  • Лимиты - установка плановых показателей
  • Анализ выполнения - контроль выполнения планов

Запросы

  • Произвольный запрос - выполнение SQL-запросов

Отчеты

  • Отчет по добыче - сводная информация о добыче
  • Отчет по затратам - анализ расходов
  • Отчет по лимитам - выполнение плановых показателей

3. Последовательность работы

  1. Начните с заполнения справочников в следующем порядке:
    1. Должности
    2. Участки
    3. Работники
    4. Уголь
  2. Внесите данные о добыче за текущую смену
  3. Зафиксируйте затраты ресурсов
  4. Ведите табель учета рабочего времени
  5. Установите плановые лимиты на месяц
  6. Формируйте отчеты для анализа

4. Типовые отчеты

Отчет по добыче

Содержит информацию об объеме добычи по участкам, маркам угля и периодам.

Анализ выполнения плана

Сравнение плановых и фактических показателей с вычислением процента выполнения.

Отчет по затратам

Анализ расходования электроэнергии и топлива по участкам.

5. Решение проблем

Ошибка подключения к базе данных

Проверьте:

  • Запущен ли сервер MySQL
  • Правильность настроек в config.py
  • Наличие прав доступа к базе данных

Данные не отображаются в таблицах

Нажмите кнопку "Обновить" или проверьте наличие данных в базе.

Ошибка при сохранении данных

Проверьте обязательные поля и правильность форматов ввода.
